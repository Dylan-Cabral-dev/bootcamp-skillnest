### Clase 3 — Análisis de Tiempos de Entrega
Traducción de una pregunta de negocio a métricas concretas de datos.
Análisis de volumen de envíos, peso total y tiempos de entrega (mín, máx, promedio, mediana).
Identificación de outliers y justificación de la mediana como métrica representativa.
